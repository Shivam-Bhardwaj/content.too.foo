About This Project

100 Topics 90 Seconds Each Hindi Channel

What is this?

This is the production database for 100 educational short-form videos (90 seconds each) for a Hindi YouTube channel. Each topic has a complete storyboard with shot-by-shot breakdowns, narration in English and Hindi, and AI prompts for generating visuals.

The website serves two purposes:

  • Production Tracker — Track progress from Planned → In Progress → Published
  • Reading Material — Browse narrations, storyboards, and learn from the content like a blog

1 Refine Content

Work on prompts and narration using Claude Code in WSL. Run the dev server for live preview with hot reloading.

npm run dev

Edit storyboards, refine English/Hindi narration, polish Midjourney/Runway prompts. View changes instantly in browser.

2 Generate Assets

Once content is finalized, generate assets in this order:

1. Cover Images

Midjourney thumbnails first — makes website engaging immediately

2. Voice Narration

Generate audio for English and Hindi versions

3. Scene Images

Midjourney key frames for each shot in the storyboard

4. Video Clips

Runway animations from the key frames

3 Asset Storage

Assets are stored on a dedicated server, organized by topic:

assets/ ├── 001/ │ ├── midjourney/ → cover.png, shot_01.png, shot_02.png... │ ├── runway/ → shot_01.mp4, shot_02.mp4... │ ├── voice/ → narration_en.mp3, narration_hi.mp3 │ └── final/ → final_youtube.mp4, final_instagram.mp4 ├── 002/ └── ...

4 Update Status

Track progress and add social links when published:

node scripts/update-status.js 1 in_progress node scripts/update-status.js 1 published --youtube URL --instagram URL

Unit Conversions (Hindi)

Hindi narration automatically converts units for Indian audience:

Currency

$X million → ₹X crore (at 90₹/$)

Distance

Miles → Kilometers, Feet → Meters

Weight

Pounds → Kilograms

Temperature

Fahrenheit → Celsius

How to Use This Site

  • Browse topics — Use category and status filters to find what you need
  • View storyboards — Click any card to see shot-by-shot breakdown
  • Read narrations — English and Hindi side by side for each shot
  • Copy prompts — One-click copy for Midjourney and Runway
  • Export — Download prompts as text files or full JSON
  • Track progress — Filter by Published/In Progress/Planned
